# optiMES REST API

REST API App untuk aplikasi JVT OptiMES berbasis NestJS, Prisma, PostGresQL

## Getting started

### 1. Clone this repo and navigate into the project directory

```
git clone https://github.com/kufayeka/jvt_optimes_api
```

Then, navigate into the project directory:

```
cd ./jvt_optimes_api
```

Install npm dependencies:

```
npm install
```

### 2. Create, Migrate and seed the database

```
npx prisma migrate dev
```
```
npx prisma db seed
```
